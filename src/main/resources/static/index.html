Needle-in-a-Haystack Test Data

A long document (Pride and Prejudice, a long novel) acts as the haystack. We inject a few deliberately incorrect facts — the needles — at evenly spaced locations throughout the document. Ask your RAG or LLM a question, then verify whether it answers using the document or its pre-trained knowledge.

Example — a model already knows The capital of France is Paris. We inject The capital of France is New York. A document-grounded system should answer New York. If it answers Paris, it relied on its training instead of the provided document.

Choose a document size that matches your model's context window. The complete novel is approximately 478 pages.
