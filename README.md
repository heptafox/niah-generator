# niah-generator

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

A Spring Boot 4.1.0 (Java 21) application for generating Needle-In-A-Haystack
(NIAH) evaluation datasets for LLM benchmarking.

## Build & Run

```bash
./mvnw clean package      # build
./mvnw spring-boot:run    # run
./mvnw test               # test
```
